### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录 
2. 修改密码
3. 个人信息
4. 公告管理
5. 销售订单管理
6. 薪资管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（部门管理、公告类型管理、收支类型管理、销售类型管理、职位管理）
4. 公告管理
5. 收支管理
6. 销售订单管理
7. 薪资管理
8. 员工管理
